# 🏦 Mini Banking Application (CLI Based)

## 📌 Overview
This is a Command Line Interface (CLI) based banking system developed using Java.  
It allows users to perform basic banking operations like account creation, deposit, withdrawal, and balance checking.

---

## 🚀 Features
- 🧾 Create new account  
- 💰 Deposit money  
- 💸 Withdraw money  
- 📊 Check account balance  
- 🔐 Basic validation for transactions  

---

## 🛠 Tech Stack
- Java  
- OOP Concepts  
- CLI (Command Line Interface)
- MySql

---

## ▶️ How to Run

1. Clone the repository  
```bash
git clone https://github.com/iMrYaduvanshi/Mini-Banking-Application

2. Navigate to folder
    cd Mini-Banking-Application

3. Compile the code
    javac Main.java

4. Run the program
    java Main
